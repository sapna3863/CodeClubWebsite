What We Use

At Code Club, we provide students with the latest tools to help them learn and create. Here are some of the resources we use in our workshops:

Scratch

A beginner-friendly programming language where students can create interactive stories, games, and animations.

Microbits

A small, programmable computer used to teach students about electronics and coding through hands-on projects.

Jimu Robots

Robotics kits that allow students to build and program their own robots, encouraging problem-solving and engineering skills.

Microsoft MakeCode

A block-based programming platform that makes it easy for students to get started with coding and create their own interactive projects.

Arduino Uno

A popular open-source electronics platform that enables students to build everything from simple circuits to complex robotic systems.

JavaScript

One of the most widely used programming languages, essential for building websites and developing interactive content.

Python

A versatile programming language perfect for beginners. Students learn text-based coding through fun projects like games and simple applications.

Hour of Code

Fun, interactive coding activities designed to introduce students to computer science through games and puzzles featuring popular characters.
